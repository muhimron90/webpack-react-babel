#### Lets Start !

- cloning repo :  *git clone --depth=1 https://github.com/muhimron90/webpack-react-babel.git*
- move to directory : cd "NameDir"
- Run App.
  
  Dev : 
        > yarn start
        > npm run start
  Prod : 
        > yarn build
        > npm run build
  Run local-server : 
        > yarn dist-local
        > npm run dist-local
  Run Watch : 
        > yarn watch
        > npm run watch
